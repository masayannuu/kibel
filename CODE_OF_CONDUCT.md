# Code of Conduct

## Our standard

We are committed to a welcoming, harassment-free environment for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and focus on technical issues.
- Accept feedback gracefully.

## Unacceptable behavior

- Harassment, discrimination, or intimidation.
- Personal attacks, trolling, or doxxing.
- Publishing private information without consent.

## Reporting

Please report unacceptable behavior through a private GitHub Security Advisory in this repository.
Maintainers will review reports and take appropriate action.
